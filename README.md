# EmojiArt2020
EmojiArt - App for downloading images from internet and placing emoji on it.

Developed using:
- MVVM Model
- SwiftUI, Combine
- Data, URL
- FileManager
- JSON
- URLSession
- Gesture
- Binding, EnvironmentObject, State, Published etc.
- extension
- NavigationView ( with button, alert etc.)
- UIKit Integration
